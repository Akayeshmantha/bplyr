# noplyr

Run basic functions of `dplyr` and `tidyr` with only base `R` and `rlang`, for Non DBI related data manipulations.

## installation

```r
remotes::install_github('yonicd/noplyr')
```

## dplyr

  - arrange
  - count
  - filter
  - group_by
  - mutate 
  - rename
  - select
  - summarize

## tidyr

  - gather
  - spread
  - unite
